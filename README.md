# Trang Web học tập tiếng anh (by Nhóm 9)

## Installation
**1. Clone project**
```
https://github.com/ThangCRND/Web-English-Learn.git
```
**2. In BackEnd Folder**
```
cd BackEnd
npm install
node server.js
```
**#3. In FrontEnd Folder**

```
cd FrontEnd
npm install
npm run dev
```
**Admin Account**
```
tài khoản mặc định của admin đã được tạo sẵn
username: admin
password: admin123
```
